Research Projects

Selected Academic Projects

My research projects focus on Artificial Intelligence, Machine Learning, TinyML, IoT, Wireless Sensor Networks, Environmental Monitoring, and Smart City applications.

Major Research Projects

Intelligent Air Quality Monitoring System

Developed a machine learning-based framework for optimizing air quality monitoring station placement using clustering algorithms and environmental data analysis.

Technologies

Python, Machine Learning, PCA, K-Means, GIS, IoT

Personalized Indoor Air Quality Monitoring

Designed a TinyML-based personalized environmental monitoring framework capable of predicting indoor pollution exposure using wearable sensing devices.

Technologies

TinyML, TensorFlow Lite, IoT Sensors, Explainable AI

Image-Based AQI Prediction

Developed a deep learning model capable of estimating air quality from environmental images using convolutional neural networks.

Technologies

CNN, TensorFlow, OpenCV, Python

IoT-Based Smart Environmental Monitoring

Designed a scalable IoT architecture for continuous environmental sensing and real-time monitoring using wireless sensor networks.

Technologies

IoT, Wireless Sensor Networks, ESP32, Cloud Computing
